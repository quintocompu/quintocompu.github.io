Función del Procesador

El procesador (CPU, Central Processing Unit) es el componente más importante dentro del PC. Es el cerebro de todo el funcionamiento del sistema, el encargado de dirigir todas las tareas que lleva a cabo el equipo y de ejecutar el código de los diferentes programas. Muchas veces, con la ayuda del resto de componentes y periféricos.

Desde un punto de vista físico, una CPU es una estructura muy compleja que se compone de miles de millones de transistores fabricados con silicio. Estos se combinan formando puertas lógicas. Estas sirven para formar las diferentes estructuras que permiten tratar las instrucciones de manera ordenada y la ejecución del código.

¿Qué es una CPU?

Un procesador, o CPU, es un circuito lógico que responde y procesa instrucciones básicas para conducir una computadora. Una CPU es integral para las operaciones del sistema porque es responsable de interpretar la mayoría de los comandos de una computadora, como los procesos básicos de aritmética, lógica y E/S. Una CPU también puede asignar comandos a voluntad para otros chips y componentes dentro de un sistema.

Un CPU está compuesto internamente de los siguientes componentes:

  1. Núcleo: Es la unidad base que constituye a un CPU, que interpreta y ejecuta acciones. Originariamente los procesadores solo tenían uno (single core), pero actualmente lo mínimo es dos.
  2. Unidad de control: Es un circuito digital que extrae la instrucción de la memoria, la descifra y la ejecuta.
  3. Unidad aritmética lógica: Es un circuito digital que lleva a cabo las operaciones lógicas, matemáticas y aritméticas entre los datos.
  4. Unidad de coma flotante: Es un componente especializado en el cálculo de operaciones con coma flotante.
  5. Memoria caché: Es la memoria en la que se almacenan los datos que el usuario consulta con frecuencia, esto permite ganar velocidad al procesador.
  6. Registros: Es una memoria de alta velocidad que permite controlar y almacenar las instrucciones en ejecución.
  7. Controlador de memoria: Es un circuito que puede estar integrado al procesador y que regula el flujo de datos entre el procesador y la memoria.
  8. Bus: Es un sistema digital que envía y recibe datos entre los componentes.
  9. Tarjeta gráfica: Es el componente que procesa los datos de video e imagen, que puede estar incluido o no en el CPU.
